# Proteosim Course Package

Purpose of the reposetory:

This repository provides **Python-based educational tools** and exercises designed to teach concepts in **computational proteomics**. The primary focus is on **simulating realistic mass spectrometry data** and protein identification workflows for course instruction. This code is currently under **active development** and may undergo significant changes.
